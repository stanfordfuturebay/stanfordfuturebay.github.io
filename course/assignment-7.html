7.5 Assignment 7

Prepare a report analyzing GHG emissions in one location (of your choice) in the Bay Area. This can be just one ZIP code to simplify the processing. You are welcome to include as many data-driven insights as youâ€™d like, but at the minimum, you must include:

  1. Using techniques from Section 7.1, an analysis of vehicle emissions. You can use the SafeGraph Neighborhood Patterns data for June 2020 to calculate emissions of visitors to your location, or you can use the same dataset to calculate emissions of trips from your location elsewhere (which involves unpacking the JSON field for the entire June dataset then filtering to your selected CBG(s) as origin CBG(s)). Or, you can use LODES 2017 data to calculate commute emissions for your location as an origin and/or destination. State all your assumptions clearly.

  2. An analysis of building emissions, using PG&E data at the ZIP code level. Include residential and commercial electricity and gas usage. You can use June 2020 data to match Part 1 if you chose SafeGraph Neighborhood Patterns, or 2017 data to match LODES 2017 data.
